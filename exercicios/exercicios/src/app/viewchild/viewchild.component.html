ViewChild and ViewChildren

ViewChild e ViewChildren são usados para comunicação entre componentes com fim de acessar dados e metodos. Ambos são decorators usados para acessar classes de componentes filhos e as propriedades no componente pai (Semelhantemente a herança). Alem disso, com ViewChild, podemos acessar elementos html específicos ao invez de referenciar todo o componente.
